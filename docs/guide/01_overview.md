## 一、项目概述

### 1.1 项目定位

cwgo 是 CloudWeGo 生态的全栈代码生成工具，集成了 Kitex (RPC) 和 Hertz (HTTP) 框架，用于快速生成微服务脚手架代码。

**注意**：该项目已归档，不再维护，建议基于此进行定制化开发。

### 1.2 核心特性

- **多框架支持**：同时支持 Kitex RPC 和 Hertz HTTP
- **项目模板**：生成标准的 MVC 项目布局
- **数据库集成**：自动生成 GORM 模型和 CRUD 代码
- **客户端封装**：生成统一的客户端调用接口
- **配置管理**：支持多环境配置文件生成
- **文档数据库**：支持 MongoDB 代码生成
